Description des commandes
Commandes Description
/cor 'insert text' Envoie ton message sans fautes
/exp 'insert text' Donne une correction du message et décrit l'erreur faite par l'utilisateur, s'il n'y a pas d'erreur, le bot envoie un message de félicitations
/stats Envoie un message avec le nombre d'erreurs faites depuis que le bot est en ligne
/help T'affiche de l'aide à propos des commandes disponibles
